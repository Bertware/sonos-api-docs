# sonos-api-docs

Sonos api documentation for the local UPNP api and a generator to generate clients based on service discovery

[![Support me on Github][badge_sponsor]][link_sponsor]
[![github issues][badge_issues]][link_issues]

This library is in no way connected to [Sonos](//en.wikipedia.org/wiki/Sonos). It's just a generated documentation
